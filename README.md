# 個人簡歷網頁

這是一個以「少女風格」與「拉拉熊」為主題的響應式個人簡歷網頁模板。使用 HTML、CSS 與 JavaScript 原生語法製作，無需安裝額外框架即可執行。

## ✨ 網頁特色

*   **可愛視覺設計**：奶茶色系背景、漂浮動畫與熊耳造型頭像框。
*   **完整區塊**：包含個人資訊、技能展示（進度條）、經歷時間軸、作品集展示。
*   **動態效果**：
    *   滑動捲軸時內容會浮現 (Scroll Reveal)。
    *   滑鼠懸停作品卡片有 Q 彈放大效果。
    *   背景有療癒的漂浮圓點。
*   **響應式設計 (RWD)**：支援手機版瀏覽，導覽列會自動切換為漢堡選單，時間軸會調整為垂直單欄。

## 如何使用

1.  將 `index.html`、`style.css`、`script.js` 下載至同一個資料夾。
2.  直接雙擊 `index.html`，使用瀏覽器 (Chrome, Edge, Safari 等) 開啟即可預覽。

## 🛠️ 如何修改內容 (客製化)

### 1. 修改文字與連結
打開 `index.html`，搜尋對應的文字即可修改：
*   **姓名與介紹**：搜尋 `<h1>` 標籤。
*   **聯絡方式**：在 `.social-links` 區塊修改 `href="mailto:..."` 或 IG/GitHub 連結。
*   **技能數據**：在 `.skill-info` 修改百分比數值。
*   **經歷與作品**：直接修改 `<h3>` 標題與 `<p>` 內文。

### 2. 替換圖片
預設使用假圖 (`via.placeholder.com`)，請將 `<img>` 標籤內的 `src` 換成你自己的圖片路徑：
*   **大頭貼**：`<img src="你的照片.jpg" ...>`
*   **作品縮圖**：找到 `.card-img` 的 `style="background-image: url(...)"` 進行替換。

### 3. 修改配色
打開 `style.css`，在最上方的 `:root` 區塊修改顏色變數：
```css
:root {
    --primary-pink: #FFB7B2;  /* 修改主色調 */
    --bear-brown: #8D6E63;    /* 修改深色文字 */
}
```
